このサイトたちについて

このサイトはGitHub Pagesにて公開している静的サイトです。
使用用途は、このようなブログ形式で自分の知識をメモするように運用していきます。
VRChatでの改変術・htmlやcssでのコーディングメモ・その他プログラミングメモなどの完全自己満足サイトですが、何かの役に立てばいいなと思い公開しています。
なにか変な部分やここを修正した方が良い等の意見がありましたら私のTwitterにて受け付けております。
特にレイアウトが崩れているということが多いので、見つけられた方はご連絡いただくととてもうれしいです。

開発環境について

ここからは少し話がずれますが、当サイトの開発はWSL(Windows Subsystem for Linux)にて開発しています。
WindowsのPower Shell上ではなくVisual Studio Codeでコーディングを行い、ターミナルからコミットを行うという形でリリースしています。
使用環境は以下の通り・・・
WSL Distributor ID: Ubuntu
WSL Ubuntu Version: Ubuntu 24.04.1 LTS
WSL Windows Version: Windows 11 24H2
Use Server: GitHub
